🤖 Gemini RAG Chatbot with Streamlit (Prompt Engineering Demo)

A simple Retrieval-Augmented Generation (RAG) chatbot built using Google Gemini API and Streamlit, demonstrating prompt engineering and context-aware AI responses.

🚀 Features
 - 🔍 Retrieval-Augmented Generation (RAG) pipeline
 - 🤖 Powered by Google Gemini API
 - 🎯 Prompt engineering for better responses
 - 💻 Simple and interactive Streamlit UI
 - ⚡ Fast and lightweight demo app

🧠 How It Works
 - User enters a query
 - App retrieves relevant context (dummy retriever for now)
 - Combines query + retrieved info into an augmented prompt
 - Sends it to Gemini model
 - Displays AI-generated response

🛠️ Tech Stack
 - Python
 - Streamlit
 - Google Generative AI (Gemini API)
